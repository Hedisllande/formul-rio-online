Termos e Condições

Leia com atenção os termos aplicáveis à inscrição e participação nos programas do FitFormGym.

1. Aceitação dos Termos

Ao submeter o formulário, o utilizador declara que leu e aceitou os presentes termos.

2. Finalidade do Serviço

O FitFormGym disponibiliza programas de actividade física geral e actividade física adaptada.

3. Responsabilidades do Utilizador

  • Fornecer informações verdadeiras;
  • Respeitar os instrutores e restantes participantes;
  • Cumprir as normas de segurança.

4. Segurança Durante as Actividades

O participante deve informar o instrutor sobre limitações relevantes para a prática de exercício físico.

5. Contactos de Emergência

Os contactos de emergência serão utilizados apenas quando necessário para garantir a segurança do participante.

6. Alterações aos Termos

O FitFormGym reserva-se o direito de actualizar os termos sempre que necessário.

Política de Privacidade (RGPD)

Secção destacada com as informações essenciais sobre o tratamento de dados pessoais.

1. Responsável pelo Tratamento de Dados

FitFormGym
contacto@fitformgym.com

2. Dados Recolhidos

  • Nome;
  • Email;
  • Telefone;
  • Faixa etária;
  • Modalidade pretendida;
  • Horário preferencial;
  • Contacto de emergência;
  • Informações funcionais necessárias para adaptação das actividades físicas.

3. Finalidade da Recolha

  • Gestão de inscrições;
  • Organização de aulas;
  • Comunicação com os participantes;
  • Segurança;
  • Adaptação das actividades físicas.

4. Dados de Saúde

Os dados de saúde apenas são recolhidos quando estritamente necessários; são utilizados exclusivamente para adaptação segura das actividades físicas; e não são solicitados diagnósticos clínicos completos nem histórico médico detalhado.

5. Conservação dos Dados

Os dados são conservados apenas pelo período necessário à prestação do serviço.

6. Segurança da Informação

O FitFormGym adopta medidas adequadas para proteger os dados contra acessos não autorizados.

7. Direitos do Titular

  • Direito de acesso;
  • Direito de rectificação;
  • Direito de eliminação;
  • Direito de limitação;
  • Direito de oposição;
  • Direito de retirar consentimento.

8. Consentimento

A submissão do formulário constitui consentimento para o tratamento dos dados necessários à inscrição.
